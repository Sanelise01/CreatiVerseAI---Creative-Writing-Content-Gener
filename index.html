Overview

CreatiVerseAI is a next-generation AI-powered creative writing tool that helps users generate high-quality poems, stories, and literary prompts across various styles, tones, and genres. It is powered by Groq API and LLaMA 3.3 70B Versatile model for fast, human-like text generation.

Key Features

  • Multi-Genre Content Generation: Poems, stories, and literary prompts
  • LLaMA 3.3 Integration: Advanced, coherent, and human-like text
  • Fast Inference Optimization: Quick content generation
  • User-Friendly Interface: Simple Streamlit web app for instant output
